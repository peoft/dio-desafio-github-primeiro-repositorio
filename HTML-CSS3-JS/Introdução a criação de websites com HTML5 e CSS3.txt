Elemento HTML

É a base do html.

Estrutura básica 

<!DOCTYPE html> não é um elemento html - apenas informa ao navegador o tipo de documento
<html>
<head>
<meta>
<title>
<body>

Semântica

<section> lista de artigos.

<header> cabeçalho da página ou partes da página.

<article> conteúdo relevante dentro da página - arquivo de um blog.

<aside> biografia de um ator ou links relacionados - representado por uma barra lateral.

<footer> rodapé de uma página ou parte de uma página.

<h1>-<h6> não foram criados no html 5. Representam a importância de um título em uma página. Apenas um h1 por página. <h1> mais importante e o <h6> o menos.


Textos e links

	<h1> Título da página
	<h2> Título de seção
	<h3> Título de artigo
	<p> Textos maiores e mais densos. representa um parágrafo. Além de texto, representa imagens, vídeos entre outros conteúdos.
	<a>link</a> ancora - interliga vários conteúdos na web. Vários atributos. mas apenas dois: href (para onde está apontando) e target(indica como esse hiperlink será aberto). 
	
Tag img

	Sem tag de fechamento
	Dois atributos src (obrigatório) e alt (recomendado para melhor a acessibilidade - mostra um texto quando imagem não é carregada)
	
	Prática
	
	photo 100 x 100
	site unsplash - photos for everyone
	
Tags li, ul e ol

	Listas servem para agrupar uma coleção de itens (lista de ingredientes/contatos).
	
	<ul> ordem dos itens não é importante.
	<ol> a ordem dos itens é importante (números, letras ou algarismos romanos).
	<li> é um item da lista
	
	
Introdução a CSS 3 (1996)
	
	Regras de estilo para elementos ou grupo de elementos.
	O que são seletores? são apenas elementos html.
	Declaração é formada por uma propriedade e um valor.
	
	ID x Classe (pode representar qualquer elemento)
	
	Classe precedida por um ponto.
	ID é precedido por um hash e pode ser usado apenas uma vez na página.
	
	Navegador representa cada elemento html como uma caixa retangular (box model). Ela possui 4 áreas: margem, a borda, o padding e o conteúdo.
	
	As margens são os espaçamentos entre os elementos.
	As bordas circundam o padding e o conteúdo e é possível alterar a aparência (largura e cor).
	O padding é o espaçamento entre a borda e o conteúdo.
	Contéudo pode ser um texto, uma imagem ou um vídeo.
	
Border
	Largura: pixels, centímetros, milímetros.
	Cor: blue, #0000ff...
	Estilo: sólida, pontilhada, tracejada.
	
	
Border-radius
	Permite arredondar os cantos de um elemento.
	Unidade: pixel e porcentagem.
	Também possui top, right, bottom e left.
	
Estilizando textos

	font-family: altera a fonte do texto (web ou instaladas). 
	Fontes Web6-com encontram-se na maioria dos dispositivos (por ex.: verdana e arial).
	
	.post_title {
		font-family: Verdana, Arial; // Caso a primeira não funcione, a segunda entra como backup.
	}
	
	font-size: altera o tamanho do texto.
	font-style: altera a aparência do texto (fonte possui o estilo?
	font-weight: altera o peso do texto.
	text-transform: alterna o texto entre maiúsculo e minúsculo.
	text-decoration: usado para dar destaque a um texto.
	
Listas ordenadas e não ordenadas
	Alterar o marcador desta lista: list-style-type;
	list-style-image: propriedade adiciona uma imagem ao marcador;
	
Dimensão e alinhamento
	Width (elemento) - pixel ou porcentagem
	Height (elemento) - pixel ou porcentagem
	
	Max-width - referem-se ao máximo que os elementos podem ter; 
	Max-height
	
	Margin - colocar espaçamento entre elementos (auto)
	
	Text align - alinhar textos (left, right, center e justify)
